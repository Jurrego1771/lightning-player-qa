# Output Schema

## Salida mínima esperada del generador

- Ruta propuesta del spec
- Tipo de test
- Qué cubre
- Qué no cubre
- Señal primaria usada
- Justificación de la aserción
- Riesgos residuales

## Salida deseable

- Cambios sugeridos al page object o harness si faltan señales
- Actualización requerida de coverage docs
- Hallazgos de diseño si el feature tiene ambigüedad
